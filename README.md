# sw-project
코드 스니펫 기반 실시간 고객 행동 데이터 분석 플랫폼  이커머스 사이트의 '추첨 이벤트'와 같이 대규모 트래픽이 발생하는 상황에서, 사용자의 행동 데이터를 실시간으로 수집하고 분석하여 비즈니스 가치가 있는 인사이트를 제공합니다. 단순한 서버 비용으로만 여겨졌던 트래픽을 귀중한 데이터 자산으로 전환하는 것을 목표로 합니다.  
주요 기능:  -경량 코드 스니펫을 이용한 실시간 사용자 행동 데이터 수집  -Kafka 기반의 대규모 트래픽 처리가 가능한 비동기 데이터 파이프라인  -데이터 기반의 비즈니스 인사이트를 제공하는 분석 및 시각화 대시보드
